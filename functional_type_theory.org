* Goal
  Formalize the categorical machinery in this section needed to prove the
  conservativity result for Ax-theories generated from algebraic theories.

  Main targets:
  - translations between Ax-theories
  - equivalence of Ax-theories via classifying categories
  - theorem Th ~ Th(Cl(Th))
  - relatively free cartesian closed category generated by a finite-product category
  - gluing / comma category construction
  - the functor J : C -> Gl(T)
  - the conservativity theorem 4.10.7

  Conceptually, this section proves:
  adding functional structure to an algebraic theory does not create new
  ground-type equalities beyond the algebraic ones.
