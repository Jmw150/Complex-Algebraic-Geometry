* Overview

Worker-pool rotation: 3 subagents (α, β, γ) plus me as orchestrator (4 total
per box budget). Each subagent grabs the next available task from the queue
when their current task finishes. Single-file `coqc` for verification (never
`make` simultaneously, so rocq is always free for someone to compile-check).

Key invariant: when a task ships its result, dependent tasks become unblocked.
The orchestrator (me) routes the next-available agent to the highest-ROI
unblocked task.

** Proof sketch of =csum_omega_reverse=

Statement:
#+begin_src coq
forall (N d : nat), N <> 0%nat ->
  csum N (fun n => Cpow (omega N) ((N - n) * d)) =
  csum N (fun n => Cpow (omega N) (n * d)).
#+end_src

Case-split on whether =d mod N = 0=:
- If =d mod N = 0=: each summand on each side is =C1= (via =omega_pow_multiple=);
  both sides reduce to =csum N (fun _ => C1)=.
- If =d mod N ≠ 0=: both sides equal =C0=.  RHS by =csum_omega_nonzero= directly.
  LHS by the telescoping identity =(C1 - ω^d) · LHS = C0= derived as follows.
  Distribute =Cmul= over =Csub= and apply =csum_Cmul_l=, getting a sum of
  differences =ω^((N-n)·d) - ω^((N-n+1)·d)=.  Use =csum_add= + =csum_neg= to
  split.  Then =csum_succ_l= peels off the [n=0] term of the second sub-sum
  (the =ω^((N-n+1)·d)= sum) and re-indexes the rest as =csum N' g= where
  =g n = ω^((N-n)·d)=.  Use =omega_pow_period_add= to identify
  =ω^((N+1)·d) = ω^d=, and observe that the resulting expression is
  =T - T = 0= (where =T= is the LHS we are computing).  Conclude
  =LHS = Cinv D HD · (D · LHS) = Cinv D HD · C0 = C0= via apartness from
  =omega_pow_apart=.

** Proof sketch of =dft_idft_inv=

Mirrors =idft_dft_inv= structurally.  The critical inner sum is
=Σ_{n=0}^{N-1} ω^((N-n)·l + n·m)= (where =l= indexes the inner =idft=
and =m= the outer =dft= output index).  Case-split on =l vs m=:

- =l = m=: exponent simplifies to =N·m=, so each term equals =1=, sum
  is =N=.  Combined with =Cinject_Q_inv_N_mul_N=, gives =xs[m]=.
- =l ≠ m=, sub-cases:
  - =l ≤ m=: rewrite =(N-n)·l + n·m = n·(m-l) + N·l=, apply
    =omega_pow_period_add= to strip =N·l=, then =csum_omega_nonzero= on
    =Σ ω^(n·(m-l))= since =0 < m-l < N=.
  - =l > m=: rewrite =(N-n)·l + n·m = (N-n)·(l-m) + N·m=, apply
    =omega_pow_period_add= to strip =N·m=, then =csum_omega_reverse= to
    convert =Σ ω^((N-n)·(l-m))= to =Σ ω^(n·(l-m))=, then
    =csum_omega_nonzero=.
