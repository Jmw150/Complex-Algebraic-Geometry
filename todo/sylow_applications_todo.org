* Part I — Corollary 20: equivalent characterizations of unique Sylow subgroup
** DONE Formalize theorem: uniqueness/normality/characteristic equivalence
   Statement:
   Let P be a Sylow p-subgroup of G. Then the following are equivalent:
   1. n_p = 1
   2. P is normal in G
   3. P is characteristic in G
   4. every subgroup generated by p-elements is a p-group

** DONE Prove (1) -> (2)
   - conjugates of P are Sylow p-subgroups
   - uniqueness forces gPg^{-1} = P

** DONE Prove (2) -> (1)
   - any Sylow p-subgroup Q is conjugate to P
   - normality forces Q = P

** DONE Prove (3) -> (2)
   - characteristic implies normal

** DONE Prove (2) -> (3)
   - P is the unique subgroup of order p^a
   - automorphisms preserve subgroup order
   - uniqueness implies characteristic

** DONE Prove (1) -> (4)
   - each p-element lies in some Sylow p-subgroup
   - uniqueness gives all p-elements lie in P
   - generated subgroup is contained in P

** DONE Prove (4) -> (1)
   - let X be the union of all Sylow p-subgroups
   - every element of X has p-power order
   - <X> is a p-group by hypothesis
   - maximality of Sylow subgroup forces <X> = P and hence uniqueness

   `unique_sylow_equiv_normal_char_generated_by_p_elements`

** DONE Groups of order 30
   Prove:
   every group of order 30 has a normal subgroup isomorphic to Z_15

** DONE Proof plan
   - if Sylow 5 or Sylow 3 subgroup normal, product has order 15

   - contradiction since 24 + 20 > 30
   - conclude one of P or Q normal, hence both normal in subgroup of order 15
   - use order-pq result to deduce subgroup of order 15 is cyclic

** DONE Groups of order 12
   Prove:
   every group of order 12 either has a normal Sylow 3-subgroup or is isomorphic to A_4

** DONE Proof plan
   - if n_3 = 1 done
   - else n_3 = 4
   - count 8 elements of order 3
   - let P be Sylow 3-subgroup, compute N_G(P)=P since index 4
   - action on 4 Sylow 3-subgroups gives homomorphism G -> S_4
   - kernel lies in N_G(P)=P
   - P not normal, so kernel trivial
   - image order 12 inside S_4 containing 8 elements of order 3, hence image = A_4

** TODO Name suggestion

** DONE Groups of order p^2 q with p,q distinct
   Prove:
   every group of order p^2 q has a normal Sylow subgroup for p or q

** DONE Proof split

   - if p < q:
     - if n_q = 1 done

* Part IV — Order 60 and A_5
** DONE Proposition: if |G|=60 and n_5 > 1, then G is simple
   More precisely formalize supplied proposition:
   If |G| = 60 and G has more than one Sylow 5-subgroup, then G is simple.

** DONE Proof structure
   Assume H ◁ G is nontrivial proper.
   - By Sylow, n_5 = 6
   - if 5 divides |H| then H contains all 6 Sylow 5-subgroups
   - count at least 25 elements in H, forcing |H| = 30
   - but order-30 groups have normal Sylow 5-subgroup, contradiction
   - hence 5 ∤ |H|
   - possible orders of H reduce to 2,3,4,6,12
   - in each case H has a normal Sylow subgroup; replace H if needed by one of order 2,3,4
   - then G/H has order 30,20,15 and has normal subgroup of order 5
   - pull back gives proper normal subgroup with order divisible by 5, contradiction

** DONE Uniqueness of the simple group of order 60
   Prove:
   If G is simple of order 60, then G ≅ A_5

** DONE Proof steps
   - show G has no subgroup of index 2,3,4
     via action on cosets giving embedding into S_2,S_3,S_4 impossible by order
   - n_2 ≠ 3
   - if n_2 = 5:
     action on 5 left cosets of N_G(P) gives embedding G -> S_5
     simplicity rules out kernel
     image lies in A_5 since otherwise intersection with A_5 has index 2
     conclude image = A_5
   - if n_2 = 15:
     count and show distinct Sylow 2-subgroups P,Q have intersection of order 2
     let M = N_G(P ∩ Q)
     deduce 4 | |M| and |M| > 4, but M proper and cannot have index 3 or 1
     conclude |M|=12, hence index 5
     repeat coset action argument to embed G in A_5
     contradiction because in A_5, n_2 = 5 not 15
   - therefore only possibility gives G ≅ A_5
